# 🤖 AI Powered Investment Management System

A complete **Investment Management System** built with **Java JSP + MySQL + Apache Tomcat** featuring **5 AI-powered features** and a premium dark dashboard UI.

## 🚀 AI Features
| # | Feature | Description |
|---|---------|-------------|
| 1 | AI Recommendations | BUY/SELL/HOLD/REBALANCE with confidence scores |
| 2 | AI Risk Analyzer | Composite risk score 0–100 |
| 3 | AI Financial Chatbot | 18+ topic investment knowledge base |
| 4 | AI Portfolio Analyzer | Health score + sector analysis |
| 5 | AI Report Generator | Performance, Risk, Health, Transaction reports |

## 🛠️ Stack
- **Backend:** Java Servlets (Jakarta EE 10)
- **Frontend:** JSP, Bootstrap 5, Chart.js
- **Database:** MySQL 8.0 (XAMPP)
- **Server:** Apache Tomcat 10.1+
- **IDE:** NetBeans 17+

## ⚡ Quick Start
1. Import `sql/investment_ms.sql` in phpMyAdmin
2. Add `mysql-connector-j-9.0.0.jar` to `web/WEB-INF/lib/`
3. Run project in NetBeans → Tomcat
4. Login: `admin@investms.com` / `Admin@123`
